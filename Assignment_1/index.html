HTML Text Formatting Tags

Reference table for the core HTML tags used to format inline and block text.

Tag Purpose Example
<b> Stylistically offset text without conveying importance. This is bold text.
<strong> Indicates strong importance; typically renders bold. This is strong text.
<i> Stylistic offset such as idiomatic text; typically renders italic. This is italic text.
<em> Emphasized text; conveys stress emphasis. This is emphasized text.
<mark> Highlights text as marked or relevant. This is highlighted text.
<small> Renders side comments or fine print in smaller type. This is small text.
<del> Shows deleted or outdated text. This is deleted text.
<ins> Indicates inserted or added text. This is inserted text.
<u> Denotes non-textual annotation; typically underlines. This is underlined text.
<s> Represents text that is no longer accurate or relevant. This is strikethrough text.
<sub> Subscript text. H2O
<sup> Superscript text. E = mc2
<abbr> Abbreviation or acronym; use title attribute for expansion. WHO
