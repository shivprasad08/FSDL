HTML Text Formatting Tags

Reference table for the core HTML tags used to format inline and block text.

Tag Purpose Example Code Example
<b> Stylistically offset text without conveying importance. This is bold text. This is <b>bold</b> text.
<strong> Indicates strong importance; typically renders bold. This is strong text. This is <strong>strong</strong> text.
<i> Stylistic offset such as idiomatic text; typically renders italic. This is italic text. This is <i>italic</i> text.
<em> Emphasized text; conveys stress emphasis. This is emphasized text. This is <em>emphasized</em> text.
<mark> Highlights text as marked or relevant. This is highlighted text. This is <mark>highlighted</mark> text.
<small> Renders side comments or fine print in smaller type. This is small text. This is <small>small</small> text.
<del> Shows deleted or outdated text. This is deleted text. This is <del>deleted</del> text.
<ins> Indicates inserted or added text. This is inserted text. This is <ins>inserted</ins> text.
<u> Denotes non-textual annotation; typically underlines. This is underlined text. This is <u>underlined</u> text.
<s> Represents text that is no longer accurate or relevant. This is strikethrough text. This is <s>strikethrough</s> text.
<sub> Subscript text. H2O H<sub>2</sub>O
<sup> Superscript text. E = mc2 E = mc<sup>2</sup>
<abbr> Abbreviation or acronym; use title attribute for expansion. WHO <abbr title="World Health Organization">WHO</abbr>
<cite> Citation of a creative work. — HTML Standard — <cite>HTML Standard</cite>
<q> Inline quotation; browsers add quotation marks. She said, keep learning. She said, <q>keep learning</q>.
<blockquote> Block-level quotation with optional cite attribute.
A block quote example.

<code> Inline code fragment. Use <section> for sections. Use <code>&lt;section&gt;</code> for sections.

<var> Variable in a mathematical expression or code. x + y = 10 <var>x</var> + <var>y</var> = 10
<time> Machine-readable date or time value. January 5, 2026 <time datetime="2026-01-05">January 5, 2026</time>
